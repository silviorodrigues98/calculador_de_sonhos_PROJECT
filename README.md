> Simulador de investimentos em CDB que calcula o montante acumulado ao longo do tempo e descobre quanto você precisa investir para atingir sua meta de renda passiva mensal.

---

## ✨ Funcionalidades

- **📈 Cálculo de Montante** — Simule quanto seu dinheiro renderá ao longo do tempo com aportes mensais em CDB
- **🎯 Meta de Renda Passiva** — Defina quanto deseja receber por mês (ex.: R$ 1.000) e descubra o valor necessário investido
- **⏱️ Tempo para Atingir a Meta** — Calcule em quanto tempo você alcançará seu objetivo financeiro com base nos aportes atuais

## 📦 Como Executar Localmente

```bash
# Clone
git clone https://github.com/silviorodrigues98/calculador_de_sonhos_PROJECT.git
cd calculador_de_sonhos_PROJECT

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

> **Pré-requisitos:** Node.js 18+, npm
